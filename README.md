#  AI Resume Analyzer

An AI-based Resume Analyzer built using Python, NLP, Streamlit, and FastAPI.

This project analyzes resumes, extracts skills, calculates resume score, and matches resumes with job descriptions.

Feature

- Upload Resume PDF
- Extract Resume Text
- NLP Based Text Cleaning
- Skill Extraction System
- Resume Score Calculation
- Job Description Matching
- Missing Skills Detection
- FastAPI Backend
- Streamlit Web Interface

## Technologies Used

- Python
- Streamlit
- FastAPI
- NLP
- Pandas
- NLTK
- Machine Learning
